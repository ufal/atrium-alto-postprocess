Pipeline Overview

Three models are applied in sequence to convert raw OCR output into classified, clean text lines.

1 · Layout Analysis (ALTO XML only)

LayoutLMv3 uses 2D bounding-box coordinates to reorder tokens from multi-column pages into the correct reading sequence before any text processing begins.

2 · Split-Word Reconstruction

Hyphenated words split across lines (encoded as SUBS_TYPE="HypPart1" in ALTO) are merged automatically using the SUBS_CONTENT attribute or a trailing hyphen flag.

3 · Quality Classification

Every line passes through two stages implemented in text_util_langID.py:

  • Pre-filter — blanks and purely numeric/separator content are caught without GPU inference.
  • Structural detectors — four regex-based checks count tokens with strange symbols, symbol repetition, letter–digit–letter fusions, and mid-word uppercase artefacts.
  • DistilGPT2 perplexity — a supporting signal for longer lines; not used as a Trash indicator due to unreliability on short or non-English text.

Categories

  • Clear — passes all structural checks and perplexity gate.
  • Noisy — degraded but recoverable (single bad token, mid-word uppercase, or elevated PPL on long lines).
  • Trash — structurally corrupt; multiple corrupted tokens or co-occurring artefacts.
  • Non-text — numeric/separator-only content (dates, page numbers, measurements).
  • Empty — blank line.

Table Columns

  • Sym — count of tokens containing characters outside the allowed internal set (see detect_strange_symbols).
  • Upper — count of tokens with mid-word uppercase artefacts (see detect_mid_uppercase, Patterns 1–3).
  • PPL — DistilGPT2 perplexity; shown as "—" for pre-filtered lines where inference was skipped.

POST /process

Upload a file for extraction and classification.

Form Parameters

  • file — .xml (ALTO) or .txt
  • task_type — "auto" | "alto" | "text"

cURL Example

curl -X POST "http://localhost:8000/process" \
  -F "file=@page_01.xml" \
  -F "task_type=auto"

Response Schema

{
  "type":     "alto_xml",
  "filename": "page_01.xml",
  "cleaned_lines": [
    {
      "line_num":    1,
      "text":        "The quick brown fox jumps over the lazy dog.",
      "perplexity":  12.5,
      "sym_count":   0,
      "upper_count": 0,
      "category":    "Clear"
    },
    {
      "line_num":    2,
      "text":        "1956–1959",
      "perplexity":  0,
      "sym_count":   0,
      "upper_count": 0,
      "category":    "Non-text"
    }
  ]
}
